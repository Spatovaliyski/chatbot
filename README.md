## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

or run a build version:

```bash
npm run build && npm run start
# or
yarn build && yarn start
# etc
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the solution.
